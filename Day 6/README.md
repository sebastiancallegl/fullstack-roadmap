### Task

Create an HTML file that includes a form with the following specifications:

    - Name:

        - Validation: Only alphabetic characters allowed

        - Maximum length: 40 characters

        - Requirement: This field is mandatory

    - Last Name:

        - Validation: Only alphabetic characters allowed

        - Maximum length: 40 characters

        - Requirement: This field is mandatory

    - Telephone:

        - Validation: Only numeric characters allowed

    - Birthday:

        - Validation: Only valid dates allowed

    - Email:

        - Validation: Must be a valid email format

        - Requirement: This field is mandatory

    - LinkedIn URL:

        - Validation: Must be a valid URL format

    - Gender:

        - Options: Male and Female (only these two options should be available)

Buttons:

    - Submit Button: To submit the form.

    - Cancel Button: To clear all fields in the form.
